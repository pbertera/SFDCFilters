Gmail SFDC filters manager

This tool allows you to create a set of filters to easily organize GSS SFDC emails

The Filters sheet

This sheet is used to manage all the filters, every row defines a filter, each filter is composed by various columns divided in 2 sections: selectors and actions. Selectors are used to determined if Actions should be performed on the message.

Filters are checked from the top-down, all the selectors must match in order to apply the defined actions.

In case Stop is checked and all the selectors are matched the next filters are not evaluated.

The Config sheet

This defines some settings:

  • Main Filter: defines in which folder the filters should be applied
  • Batch size: defines how many email threads should be evaluated every time
  • Poll time: defines the interval in minutes between one run and onother

How to install

To install the filters you have to:

  1. Setup the Main Filter into the Config sheet
  2. Setup one or more filters into the Filters sheet
  3. Click on the GSS SFDC filters menu and then Install
